# Weather Dashboard

This Weather Forecast Application provides real-time weather information and a 5-day forecast for any city worldwide. Built using jQuery and the OpenWeatherMap API, the application fetches current weather data, including temperature (in Fahrenheit) and weather conditions, upon user request. Users can input the name of a city, and the application will display the current weather along with a forecast that updates dynamically. Temperature values are presented in Fahrenheit to cater to users familiar with the Imperial system.


## Features
Real-time weather information and 5-day forecast
Temperatures displayed in Fahrenheit
User-friendly interface with dynamic data display


## Usage
To use the application, simply enter the name of a city or zip code in the input field and submit. The application will then display the current weather information and a 5-day weather forecast for the specified location.
